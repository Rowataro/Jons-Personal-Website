## Getting Started


#### 1. Clone this repo

```
git clone 
```
#### 2. Change into the working directory

```
cd mysite
```

#### 3. Install dependencies

```
npm install
```

#### 4. Work locally
Watches for changes and serves locally on http://localhost:8080

```
npm run serve
```

#### 5. Create a production build

```
npm run build
``
